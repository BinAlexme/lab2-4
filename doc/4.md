Программа запрашивает у пользователя целое число, проверяет его корректность и сообщает, сколько в нем цифр. Если пользователь вводит exit, программа завершает работу.

Для ввода используйте input(), для постоянного запроса данных — while True. Проверяйте, что введенное значение является числом (isdigit() работает только для положительных), 
а для учета отрицательных чисел используйте num.lstrip('-').isdigit(). Для определения количества цифр используйте len().
